# Predicción de Fuga de Clientes (Customer Churn)

Este repositorio contiene un modelo de Machine Learning diseñado para identificar de forma proactiva a los clientes que tienen un alto riesgo de abandono (fuga de clientes).

##  Objetivo del Proyecto
El fin principal es predecir la pérdida de clientes.

##  Tecnologías Utilizadas
* **Lenguaje:** Python 3.10+
* **Librerías principales:** Pandas, NumPy, Scikit-Learn, TreeRegressionClassifier, Matplotlib, Seaborn.
* **Entorno:** Jupyter Notebook .


## 📈 Resultados del Modelo
El modelo final (basado en XGBoost) alcanzó las siguientes métricas en el conjunto de prueba:
* **F1-socre:** 68%
* **AUC-ROC:** 0.86
* **Recall (Fuga):** 82% (Priorizado para capturar la mayor cantidad de abandonos reales).
